# Student ID card templates

Place `front.png` and `back.png` in each level folder:

- `grade_school/`
- `high_school/`
- `college/`

After cloning, copy your designs here or run:

```bash
php artisan id-cards:init-student-templates
```

Template image files are not committed to git. See `config/id_cards.php` for layout coordinates.
